# Core_engine

这是什么:
1,更现代化的mod管理装置，把mod的下载、更新、开关、卸载、诊断全部搬进游戏侧栏。
装好一次，以后不再手动折腾zip。新mod和新版本在游戏内一键安装，缺前置会自动提醒，出问题诊断页一键清缓存自助修复。
2，可细可粗的开关自由：可以整包停用任意mod，也可以只关某一项功能（比如只关前景背景、只关经济结算），随时恢复，不删文件不丢存档。
我们持续简化着mod制作和美化制作难度，因此如果您的mod想要从我们的mod 管理器分发，或是bug反馈，请联系我们的QQ群540579363


我们的框架就是管理器本身，其是用于管理mod、分发mod、制作mod的现代化工具。

请将其置于第一个安装,并且在安装后自行更新。
